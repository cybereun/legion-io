![License](https://img.shields.io/badge/License-Custom-purple?style=for-the-badge)

**Legion.io**는 실시간 양방향 웹소켓 통신을 기반으로 하는 웹 브라우저용 대규모 RTS/군집 전투(Io-style) 게임입니다. 금화를 수집하고 군단을 키워 영토를 확장하며, 친구들과 함께 방을 만들어 실시간 멀티플레이어 배틀을 펼칠 수 있습니다.

---

## 🎮 게임 설명 및 특징

*   **실시간 멀티플레이 매칭 (방 코드 매칭):** 고유한 4자리 방 코드를 발급받아 특정 친구들과 즉시 한 전장에서 만날 수 있습니다.
*   **참가자별 고유 색상 시스템 (피아 식별):** 방에 입장하는 순서대로 중복되지 않는 고유의 색상(빨, 주, 노, 초, 파, 남, 보)을 할당받습니다. 플레이어 본체(군주)뿐 아니라, 소환하는 모든 병사 유닛과 지어진 건물의 팀 컬러 장식이 해당 색상으로 실시간 매핑됩니다.
*   **하이브리드 싱글/멀티플레이 지원:** 네트워크 연결 없이 지능적인 6명의 AI 봇들과 전투를 벌이며 연습할 수 있는 오프라인 싱글플레이 모드도 완벽히 지원합니다.
*   **RTS 영토 확장 및 기지 건설:** 병영, 마구간, 공성소를 건설하여 보병, 경기병, 기갑 기사, 투석기 등 고티어 유닛들을 자동으로 육성하고 나만의 요새를 구축할 수 있습니다.
*   **전술 미니맵(Strategic Radar):** 우측 하단에 아군 및 적군의 군주 위치, 유닛 규모, 요새 건물의 배치를 보여주는 미니맵 레이더가 탑재되어 전략적 지휘가 가능합니다.
*   **다이내믹 카메라 줌 & 타격감:** 보유한 병사의 규모에 따라 카메라의 줌아웃 레벨이 동적으로 부드럽게 조정되며, 대형 투석기 공격 시 화면 흔들림(Screen Shake) 효과가 더해져 박진감 넘치는 액션을 제공합니다.
*   **필드 버프 아이템:** 전장에 무작위로 생성되는 힐 포션(❤️), 속도 물약(⚡), 골드 자석(🧲)을 획득하여 전투 판세를 뒤집을 수 있습니다.
*   **건물 3단계 업그레이드 시스템**: 모든 건물(금광, 집, 방어타워, 유닛 생산소)은 최대 3레벨까지 단계별 업그레이드가 가능합니다. 업그레이드 시 건물 외형이 화려하게 변하고 성능(생산 속도 가속, 타워 공격력/사거리 대폭 상승, 금 수확량 증가, 인구 한도 상향 등)이 대폭 강화됩니다.

---

## 🔺 건물 3단계 업그레이드 상세
모든 건물은 골드를 소모하여 최대 3레벨까지 진화시킬 수 있습니다.

*   **업그레이드 조작**: 아군 건물을 **마우스로 클릭**하면 좌측 하단에 업그레이드 UI 카드(현재 효과, 다음 레벨 효과, 필요 금화)가 표출되며, 버튼을 눌러 실행합니다. (`ESC`나 카드 내 `닫기` 버튼 클릭 시 UI가 닫힙니다.)
*   **업그레이드 비용**:
    *   **Level 2 업그레이드**: 해당 건물 기본 비용의 **4배** (예: 금광 250G ➡️ 1,000G)
    *   **Level 3 업그레이드**: 해당 건물 기본 비용의 **8배** (예: 금광 250G ➡️ 2,000G)
*   **건물별 업그레이드 효과**:
    *   **금광 (Mine)**: 초당 금 수확량 상향 (Lv.1: +3G ➡️ Lv.2: +8G ➡️ Lv.3: +18G)
    *   **집 (House)**: 인구 한도 기여도 상향 (Lv.1: +15 ➡️ Lv.2: +35 ➡️ Lv.3: +65)
    *   **아처 타워 / 메이지 타워**: 데미지, 공격 사거리, 공격 쿨타임(사격 속도)이 대폭 강화됩니다. 3레벨 도달 시 아처타워는 거대 발리스타로, 메이지타워는 공전 크리스탈 파편과 성핵을 가진 대마법 요새로 진화합니다.
    *   **병영 / 마구간 / 공성소**: 유닛 자동 생산 및 훈련 소요 시간이 대폭 단축됩니다. (2레벨 시 25% 단축, 3레벨 시 50% 단축)
*   **멀티플레이 동기화**: 플레이어가 건물을 업그레이드하면 같은 방의 다른 플레이어 화면에서도 실시간으로 건물의 레벨, 바뀐 외형, 체력, 특수 파티클 이펙트가 완벽히 동기화됩니다.
*   **AI 봇(적군) 인공지능 강화**: 싱글플레이 오프라인 모드에서도 적 AI 군주들이 골드를 모으면 15%의 확률로 자신의 건물을 요새화하기 위해 업그레이드를 실행합니다.

## ⚔️ 멀티플레이어 참여 가이드

1.  브라우저를 열고 게임 페이지에 접속합니다. (개발 실행 시 기본 주소: `http://localhost:8000`)
2.  닉네임을 적고 **[대시보드 시작]**을 누릅니다.
3.  **방장(Host):** **[방 개설하기 (Host)]** 버튼을 클릭하여 방을 개설한 후, 화면 우측 대기실 카드에 생성된 4자리 방 코드를 친구들에게 알려줍니다.
4.  **참가자(Guest):** 방 코드 입력창에 발급받은 코드를 정확히 입력한 뒤 **[방 참가]**를 누릅니다.
5.  대기실 인원 목록을 확인한 후, 방장이 **[게임 시작]**을 누르면 실시간 동기화 전장에서 배틀이 펼쳐집니다.

## 🛠️ 개발 및 빌드 환경 가이드

본 프로젝트는 Node.js와 Socket.io 패키지를 사용하며, 구글 드라이브(Z:) 등의 가상 드라이브에서 I/O 지연으로 인한 패키지 손상이 발생할 경우 로컬 런타임에서 실행하는 것을 권장합니다.

```bash
# 의존성 모듈 설치
npm install

# 실시간 멀티플레이어 서버 실행
node server.js
```
서버 실행 후, 웹 브라우저를 열고 `http://localhost:8000`에 접속하여 플레이하실 수 있습니다.
